El Tiempo en Madrid - Drupal 8 Module
Shows an Icon with the current weather conditions in Madrid Spain.

INSTALATION
---------------------
Install as usuall,  like any Drupal module.

USAGE
----------------------

Once the module is installed, go to /admin/structure/block and add the block
called 'Icono de Estado del tiempo' to any region that you like.
The block should be visible showing the Time in Madrid Spain.
